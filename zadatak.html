Uvod u Front-end Development

Front-end Development se bavi razvojem web strana i web aplikacija.

HTML i CSS

HTML (Hypertext Markup Language) defenise osnovnu strukturu web strana.
Funkcioniše tako što podatke strukturira korišćenjem oznaka za markiranje.

CSS (Cascading Style Sheets) je jezik za stilizovanje,
uređuje izgled i sluzi za formatiranje svih elemenata u HTML dokumentu.

JS

JS (Java Script) je programski jezik koji sluzi da ucini stranicu interaktivnom,
takodje se koristi za kreiranje web igra, animacija i rukije podacima na klijentskom delu web servera.
